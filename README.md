# IIST-LAB-Computer-Networks
Contains the C programs written for the lab exercises of AV341: Computer Networks Lab (2022).
Files beginning with "udp" are parts of Lab3 exercises. UDP was used to exchange packets between client and server. Average round trip time was found.
Files beginning with "tcp" are parts of Lab5 exercises. TCP was used to exchange packets between client and server. Number of receive system calls was recorded.
